## üí° Tips for Success

- **Read Carefully**: Questions may have subtle distinctions
- **Process of Elimination**: Rule out obviously wrong answers
- **Knowledge Area Awareness**: Use topic hints to recall concepts
- **Time Management**: Don't spend too long on any one question
- **Review Flagged Items**: Mark difficult questions for review

## üîß Troubleshooting

### Common Issues

**Issue**: Quiz won't start
- **Fix**: Install tkinter (see QUICK_START.md)

**Issue**: File not found errors
- **Fix**: Ensure all .md files are in same directory as .py file

**Issue**: Questions seem repetitive
- **Fix**: This is expected - restart app for new random order

**Issue**: All answers are "B"
- **Fix**: Answer positions are randomized in GUI; correct answers vary
